# Scoping-Invoice-Generator
Generates formatted invoices from a single excel sheet that includes, reporter names, job titles, dates, number of pages, as well as the rate in $/page. 

## **WARNING:** This generator works for only the specific columns in a singular excel sheet:
    1. Reporter Name
    2. Job Title
    3. Job Date
    4. Number of Pages
    5. Rate (in $/Page)
Future formats may be added if requested
