# Scoping Invoice Generator
Generates formatted invoices from a single excel sheet that includes, reporter names, job titles, dates, number of pages, as well as the rate in $/page. 

### **WARNING:** This generator works for only the specific columns in a singular excel sheet:
1. Reporter Name
2. Job Title
3. Job Date
4. Number of Pages
5. Rate (in $/Page)
Future formats may be added if requested

## Setup Instructions:
1. Click on the latest release listed under 'Releases' (located on the right)
2. Download the .zip file 
3. Once downloaded, unzip the file.
4. In the unzipped folder, open 'header.json'
5. Fill out fields with your (or your company's) information
6. Save the file
*OPTIONAL: right-click on the .exe folder and click 'create shortcut'. Move this to the desktop for easier access
